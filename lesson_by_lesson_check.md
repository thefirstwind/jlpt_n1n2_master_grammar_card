## N1 第1课 (n1-l01)

### 语法点1: 〜が早いか (n1-l01-g01)

**题目1**: 落書きをしていた子供たちは、わたしの顔を（　）が早いか、逃げていった。
- a 見て
- b 見る
- c 見るの

**分析**: 〜が早いか接动词辞书形/た形。这里应该是"見る"（辞书形）。
**正确答案**: b ✓ (HTML显示: data-answer="b")

---

**题目2**: 新しい化粧品が発売されるが早いか、（　）。
- a 若い女性たちが殺到した
- b 多くの女性に買ってほしい
- c すぐに買ってください

**分析**: 〜が早いか后接事实描述。b是希望表达，c是命令表达，都不合适。a是事实描述。
**正确答案**: a ✓ (HTML显示: data-answer="a")

---

**题目3**: 新幹線が走り出すが早いか、（　）。
- a みんなでお弁当を食べよう
- b みんなとても楽しそうだった
- c 彼はビールを飲み始めた

**分析**: a是意向表达（食べよう），不符合语法。b是状态描述（楽しそうだった），c是事实描述（飲み始めた）。但b更自然，符合"一...就..."的意外感。
**正确答案**: b ✓ (HTML显示: data-answer="b")

## N1 第2课 (n1-l02)

### 语法点1: 〜を皮切りに（して）・〜を皮切りとして (n1-l02-g01)

**题目1**: その宇宙飛行士の講演会は（　）を皮切りに、130か所で行われた。
- a 北海道
- b 外国
- c 全国

**分析**: 表示"以...为开端"。从具体地点开始合理。a"北海道"正确。
**正确答案**: a ✓ (HTML显示: data-answer="a")

---

**题目2**: この選手は昨年の地区大会入賞を皮切りに、（　）。

- c いくつもの大会で好成績を残している

**分析**: 表示连续成功。c"在多个比赛中取得好成绩"符合。
**正确答案**: c ✓ (HTML显示: data-answer="c")
