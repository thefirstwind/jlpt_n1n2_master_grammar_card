### n2-l14-g01 〜にもかかわらず

**规则**：意味： ⇒ 〜には影響されないで、あることをする。

#### Q1 彼は両親が（　）にもかかわらず、毎日遊び歩いている。


| 选项           | 分析                              |
| ------------ | ------------------------------- |
| a 入院         | 不合本语法点或语境                       |
| b 入院中だ       | 不合本语法点或语境                       |
| **c 入院中である** | 名詞普通形须用「である」；「入院中である」再接にもかかわらず。 |


**答案：c**

#### Q2 「うるさい」と注意されたにもかかわらず、みんな（　）。


| 选项               | 分析                            |
| ---------------- | ----------------------------- |
| **a おしゃべりをやめない** | 被提醒安静后仍不停说话，与常识相反→おしゃべりをやめない。 |
| b 話している          | 不合本语法点或语境                     |
| c しばらく静かにした      | 不合本语法点或语境                     |


**答案：a**

#### Q3 このような大雨にもかかわらず、（　）。


| 选项                    | 分析                            |
| --------------------- | ----------------------------- |
| a 皆さん、集まってください        | 不合本语法点或语境                     |
| b みんなで集まりましょう         | 不合本语法点或语境                     |
| **c ようこそお集まりくださいました** | 大雨仍出席欢迎→ようこそお集まりくださいました（事实句）。 |


**答案：c**
