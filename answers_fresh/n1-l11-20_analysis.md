# 第11课

## n1-l11-g01 〜べく

**规则**：動辞書形＋べく；后接意志行为；前后主语一致；不接働きかけ

### Q1 中学校の同窓会に（　）べく、わたしはふるさとに帰った。


| 选项         | 分析     |
| ---------- | ------ |
| a 参加し      | 连用形不接  |
| **b 参加する** | 辞書形＋べく |
| c 参加したい    | たい形不接  |


**答案：b**

### Q2 アルバイトを始めるべく、（　）。


| 选项                 | 分析        |
| ------------------ | --------- |
| **a 彼は必要な書類をそろえた** | 同一主语的意志行为 |
| b 必要な書類がそろった       | 无意志/主语不一致 |
| c 必要な書類は何ですか       | 働きかけ      |


**答案：a**

### Q3 これを田中さんに知らせるべく、メールを（　）。


| 选项           | 分析        |
| ------------ | --------- |
| **a 書いた**    | 意志行为「書いた」 |
| b 書いておいてください | 请求        |
| c 書いたほうがいいです | 建议        |


**答案：a**

### Q4 小林氏は今度の選挙に立候補するべく、（　）。


| 选项             | 分析            |
| -------------- | ------------- |
| a 党がそれを認めた     | 主语变为党         |
| **b 手続きを済ませた** | 手続きを済ませた=意志行为 |
| c 党の応援が必要だった   | 状态非行为         |


**答案：b**

## n1-l11-g02 〜んがため（に）

**规则**：動ない形＋んがため；する→せん；重大目的；主语一致

### Q1 わが子の無罪を（　）がため、母親は必死で証拠を探した。


| 选项         | 分析          |
| ---------- | ----------- |
| **a 証明せん** | 証明せん=するの未然形 |
| b 証明する     | 普通形不接       |
| c 証明しよう    | 意志形不接       |


**答案：a**

### Q2 戦争反対という思いを示さんがために、（　）。


| 选项                 | 分析         |
| ------------------ | ---------- |
| a どんな方法があるのか       | 疑问         |
| b さまざまな方法がある       | 一般陈述       |
| **c 彼はあらゆる方法を試みた** | 彼は試みた=意志行为 |


**答案：c**

### Q3 権力を保たんがために、彼は（　）。


| 选项             | 分析       |
| -------------- | -------- |
| **a 強硬手段を取った** | 強硬手段を取った |
| b 実力があった       | 能力描写     |
| c 何もできなかった     | 否定结果     |


**答案：a**

## n1-l11-g03 〜をもって

**规则**：名詞＋をもって=以…为手段；をもってすれば=凭…之力

### Q1 当選者には5日以内に賞品をお送りします。（　）をもって当選発表といたします。


| 选项          | 分析                 |
| ----------- | ------------------ |
| a 宅配便       | 具体工具               |
| **b 賞品の発送** | 賞品の発送をもって=以发货为手段宣布 |
| c 賞品        | 赏品非手段              |


**答案：b**

### Q2 森田さんは（　）をもって病気を克服したのである。


| 选项           | 分析       |
| ------------ | -------- |
| **a 人一倍の努力** | 抽象手段「努力」 |
| b 新開発の薬      | 具体药物偏日常  |
| c 健康的な食事     | 同上       |


**答案：a**

### Q3 彼の実力をもってすれば、（　）。


| 选项                | 分析      |
| ----------------- | ------- |
| a 今大会での優勝は無理だろう   | 与实力评价矛盾 |
| **b 成功は間違いないだろう** | 凭实力必成功  |
| c やはり不安が残る        | 与用法矛盾   |


**答案：b**

## n1-l12-g01 〜ばこそ

**规则**：〜ばこそ=正因为…才；后多接のだ；强调反直觉理由

### Q1 内容が高度（　）こそ、わかりやすく説明する必要がある。


| 选项         | 分析        |
| ---------- | --------- |
| a だったらば    | だったらば不通   |
| **b であれば** | 名詞＋であればこそ |
| c であったらば   | であったらば不通  |


**答案：b**

### Q2 忙しければこそ、（　）のだ。


| 选项                 | 分析                |
| ------------------ | ----------------- |
| a ご飯を食べる時間もない      | 消极，与こそ强调弱         |
| b 旅行には行けない         | 单纯不能旅行            |
| **c 時間の使い方がうまくなる** | 正因为忙才学会善用时间（积极结果） |


**答案：c**

### Q3 収入が少なければこそ、（　）のだ。


| 选项                | 分析            |
| ----------------- | ------------- |
| a 暮らせない           | 暮らせない太消极直因    |
| b 楽しくない           | 楽しくない         |
| **c 楽しく暮らす努力をする** | 正因为收入少才努力快乐生活 |


**答案：c**

### Q4 わたしたち二人の関係が良ければこそ、（　）。

**答案：a**

### Q5 あなたのことを思えばこそ、（　）。

**规则**：〜を余儀なくされる=被迫；させる=使被迫

### Q1 外国への転勤で、父は（　）を余儀なくされた。


| 选项         | 分析           |
| ---------- | ------------ |
| a 困惑       | 困惑           |
| b 母に手伝い    | 帮母亲          |
| **c 単身赴任** | 単身赴任を余儀なくされた |


**答案：c**

### Q2 ダム建設はその地区の住民に（　）を余儀なくさせることになる。


| 选项       | 分析         |
| -------- | ---------- |
| a 反対運動   | 反対運動       |
| **b 移転** | 移転を余儀なくさせる |
| c 便利な生活  | 便利生活       |


**答案：b**

### Q3 企業の海外進出が国内産業の衰退を（　）。


| 选项              | 分析           |
| --------------- | ------------ |
| **a 余儀なくさせている** | 衰退を余儀なくさせている |
| b 余儀なくされている     | 被动           |
| c 余儀なくさせられている   | 双重被动         |


**答案：a**
