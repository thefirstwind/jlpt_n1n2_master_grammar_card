# 第6课

## n1-l06-g01 〜んばかりだ

**规则**：動ない形 + んばかり（だ/に/の）；まるで〜しそうなほど。する→せん。

### Q1 うちの犬は、自分もこの家の家族の一人だと（　）の顔をしてテレビを見ている。


| 选项           | 分析                            |
| ------------ | ----------------------------- |
| a 言うばかり      | 言うばかり：肯定形、接続不自然               |
| **b 言わんばかり** | 言わんばかり：ない形＋んばかり，「まるで言いたげな顔」 ✓ |
| c 言わないばかり    | 言わないばかり：形不对                   |


**答案：b**

### Q2 ニュースを聞いて、彼は（　）驚いた。


| 选项               | 分析                         |
| ---------------- | -------------------------- |
| **a 飛び上がらんばかりに** | 飛び上がらんばかりに：飛び上がりそうなほど驚いた ✓ |
| b 飛び上がると言わんばかりに  | と言わんばかり：冗長、慣用外             |
| c 飛び上がれんばかりに     | 飛び上がれんばかり：可能形不对            |


**答案：a**

### Q3 今にも（　）子を置いて、母親はどこへ行ってしまったのだろう。


| 选项              | 分析                  |
| --------------- | ------------------- |
| a 泣き出さんばかりに     | 泣き出さんばかりに：副詞句，修饰动词  |
| **b 泣き出さんばかりの** | 泣き出さんばかりの：名詞「子」修饰 ✓ |
| c 泣き出さんばかりな     | 泣き出さんばかりな：语尾错误      |


**答案：b**

### Q4 彼女はその人の死を知って、（　）声を上げて泣いた。


| 选项                 | 分析                    |
| ------------------ | --------------------- |
| a みんなに聞こえんばかりの     | 聞こえんばかり：可通但不如 c 贴切    |
| b 涙があふれんばかりの       | 涙があふれんばかり：与「声を上げて」不搭  |
| **c のどが張り裂けんばかりの** | のどが張り裂けんばかりの：声嘶力竭地哭 ✓ |


**答案：c**

## n1-l06-g02 〜とばかり（に）

**规则**：発話文 + とばかり（に）；无声而以言行事。

### Q1 息子は「入るな」とばかりに、（　）。


| 选项                  | 分析                  |
| ------------------- | ------------------- |
| a 大声で言った            | 大声で言った：与「声は出さない」矛盾  |
| b 部屋を出ていってしまった      | 部屋を出る：与「入るな」の意味不合   |
| **c 部屋にかぎをかけてしまった** | かぎをかける：以行动表示「入るな」 ✓ |


**答案：c**

### Q2 彼女は平凡なのはつまらないとばかりに、（　）。


| 选项               | 分析                      |
| ---------------- | ----------------------- |
| **a 変わった服を着ている** | 変わった服：以打扮表现「平凡はつまらない」 ✓ |
| b 楽しそうにしている      | 楽しそう：与批判平凡相反            |
| c 面白い人だ          | 面白い人だ：叙述性格，非態度          |


**答案：a**

### Q3 その女の子は皿の上の野菜を見て、（　）とばかり、横を向いた。


| 选项             | 分析                |
| -------------- | ----------------- |
| a 「おいしそう」      | 「おいしそう」：与横を向く相反   |
| **b 「食べたくない」** | 「食べたくない」：无声拒绝蔬菜 ✓ |
| c 「わあ、いっぱい」    | 「わあ、いっぱい」：欢喜，不合   |


**答案：b**

## n1-l06-g03 〜ともなく・〜ともなしに
