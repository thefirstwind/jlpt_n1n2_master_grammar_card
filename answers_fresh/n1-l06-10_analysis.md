# 第6课

## n1-l06-g01 〜んばかりだ

**规则**：動ない形 + んばかり（だ/に/の）；表示“几乎像要……一样”，强调状态逼真到仿佛下一秒就会发生。`する` 要变成 `せん`。

### Q1 うちの犬は、自分もこの家の家族の一人だと（　）の顔をしてテレビを見ている。

| 选项 | 分析 |
| --- | --- |
| a 言うばかり | 这里缺少「ん」而且是普通的「言うばかり」，读成“只是说……”一类意思，和题干要表达的“仿佛在说自己也是家人”不合。 |
| **b 言わんばかり** | `言わんばかり` 是「言う」的未然形 + `んばかり`，表示“脸上简直像要说出来一样”。 |
| c 言わないばかり | 这个形式不是 `んばかり` 的正确接法，既不是固定表达，也没有“几乎要说……”的逼真感，因此不成立。 |

**答案：b**

### Q2 ニュースを聞いて、彼は（　）驚いた。

| 选项 | 分析 |
| --- | --- |
| **a 飛び上がらんばかりに** | `飛び上がらんばかりに` 表示“惊讶到简直要跳起来一样”，比单纯说“很惊讶”更强，完全符合 `んばかり` 夸张逼真的语感。 |
| b 飛び上がると言わんばかりに | 这里把「飞起来」又套进「と言わんばかり」里，结构上很别扭；`と` 不是这个句型要的核心接法，读起来冗长。 |
| c 飛び上がれんばかりに | `飛び上がれ` 是命令/可能混杂的形式，不是 `んばかり` 要求的未然形接续，所以语法不对。 |

**答案：a**

### Q3 今にも（　）子を置いて、母親はどこへ行ってしまったのだろう。

| 选项 | 分析 |
| --- | --- |
| a 泣き出さんばかりに | 这是副词性的说法，更适合放在动词前面修饰行为；但空格后面接的是名词 `子`，这里需要名词修饰语。 |
| **b 泣き出さんばかりの** | `泣き出さんばかりの子` 是“眼看就要哭出来的孩子”，`の` 可以直接修饰 `子`，结构最自然。 |
| c 泣き出さんばかりな | `な` 不能这样接在这里，`んばかり` 后接名词时通常用 `の`，因此这个形式不对。 |

**答案：b**

### Q4 彼女はその人の死を知って、（　）声を上げて泣いた。

| 选项 | 分析 |
| --- | --- |
| a みんなに聞こえんばかりの | 这个词组虽然也能说得通，但重点只是“声音很大”；题干里更想表达的是“悲痛到喉咙都快撕裂一样”的强烈状态，所以力度不够。 |
| b 涙があふれんばかりの | `涙があふれんばかり` 更像描述“眼泪快要溢出来”，但后面接的是 `声を上げて泣いた`，和“声音爆发”的重点不完全贴合。 |
| **c のどが張り裂けんばかりの** | `のどが張り裂けんばかりの声` 很强烈地写出“嗓子都快裂开了”的哭喊状态，和后面的 `声を上げて泣いた` 完全呼应。 |

**答案：c**

## n1-l06-g02 〜とばかり（に）

**规则**：`発話文 + とばかり（に）`；不直接说出口，而是通过动作、表情、态度来“像是在说……”

### Q1 息子は「入るな」とばかりに、（　）。

| 选项 | 分析 |
| --- | --- |
| a 大声で言った | 题干前面已经有「入るな」这句“仿佛在说”的内容，所以后面不该再变成“他大声说了”；前后逻辑打架。 |
| b 部屋を出ていってしまった | `入るな` 的意思是“不要进来”，而“离开房间”不能直接等同于“别进来”，表达方向不够准。 |
| **c 部屋にかぎをかけてしまった** | 锁门是不用说出口、直接用行动表达“别进来”的典型方式，正好符合 `とばかりに` 的“无声地示意”语感。 |

**答案：c**

### Q2 彼女は平凡なのはつまらないとばかりに、（　）。

| 选项 | 分析 |
| --- | --- |
| **a 変わった服を着ている** | 她用穿得与众不同来表现“平凡太无聊了”，这就是不说出口、直接用行为表达态度。 |
| b 楽しそうにしている | 只是“看起来很开心”，没有把“平凡无聊，所以要特意做不同的事”这个态度表现出来。 |
| c 面白い人だ | 这是对人的评价，不是动作或态度，和 `とばかりに` 需要的“动作显示态度”不搭。 |

**答案：a**

### Q3 その女の子は皿の上の野菜を見て、（　）とばかり、横を向いた。

| 选项 | 分析 |
| --- | --- |
| a 「おいしそう」 | 她横过脸去，并不是在表达“看起来很好吃”；这和“拒绝蔬菜”的动作方向相反。 |
| **b 「食べたくない」** | 没有把“不想吃”直接说出来，而是通过把脸转开来表示拒绝，这正是 `とばかり` 的典型用法。 |
| c 「わあ、いっぱい」 | 这是带有兴趣或惊讶的态度，和“把脸转开表示不想吃”完全相反。 |

**答案：b**

## n1-l06-g03 〜ともなく・〜ともなしに

**规则**：`A 辞書形 + ともなく` 表示无意识地做某动作；`疑问词 + ともなく` 表示“也不知是哪里/什么时候……”这种不特定感。

### Q1 （　）ともなく空を見ていたら、珍しい鳥が目に入った。

| 选项 | 分析 |
| --- | --- |
| **a 見る** | `見るともなく` 表示“无意识地看着天空”，刚好符合“并没有特意去看，只是顺手看着”的感觉。 |
| b 見よう | `見よう` 是意志形，带着“要看”的明确目的，和“无意识地”完全相反。 |
| c 見ている | `見ている` 是进行状态，但这里需要的是“并没有特意去看”的起点形式，所以不合适。 |

**答案：a**

### Q2 （　）ともなしにラジオをつけておくのが好きだ。

| 选项 | 分析 |
| --- | --- |
| a 何でも聞く | 这是“什么都听”的一般说法，不是 `何を聞くともなしに` 这种“不特意挑着听”的感觉。 |
| **b 何を聞く** | `何を聞くともなしに` 是“也不知道要听什么，就随手把收音机打开”的自然表达。 |
| c ピアノ曲を聞く | 这个说法太具体了，失去“不特定、随手听”的语感，不像 `ともなしに`。 |

**答案：b**

### Q3 父は日曜日、どこへ行くともなしに（　）。

| 选项 | 分析 |
| --- | --- |
| a うちにいた | 和“去了哪里也不知道”矛盾，`ともなしに` 的意思就是没目标地走动，不可能还待在家里。 |
| **b 一人で出かけた** | “没有明确目的地，一个人出门了”正好对应 `どこへ行くともなしに` 的漫无目的感。 |
| c 考えた | 这里只说“想了”，没有表现“往外走、但没有目的”的动作链。 |

**答案：b**

### Q4 （　）わたしは彼を尊敬するようになった。

| 选项 | 分析 |
| --- | --- |
| a 何ともなく | 这个词没有“也不知何时开始”的意思，和后面的“慢慢产生敬意”接不上。 |
| b だれともなく | `だれともなく` 是“不知是谁”，不是“不知何时”；语义点错了。 |
| **c いつからともなく** | `いつからともなく` 就是“从不知何时起，渐渐地”，非常适合后面“开始尊敬他”这种慢慢形成的变化。 |

**答案：c**

### Q5 さっきまであの木の枝に鳥が数羽いたが、（　）飛んでいってしまった。

| 选项 | 分析 |
| --- | --- |
| **a どこへともなく** | `どこへともなく` 是“不知飞到哪里去了”，正好符合鸟突然不知去向的场景。 |
| b どこともなく | 缺少助词 `へ`，结构不完整，不能这样用。 |
| c 何羽ともなく | `何羽` 是数量，不是去向，和这里“飞到哪里去了”的意思不合。 |

**答案：a**

## n1-l06-g04 〜ながらに（して）

**规则**：`ます形/名 + ながらに（して）`；表示“保持某种状态不变地……”或固定搭配。

### Q1 大川選手は走るのも速いしボールの扱いもうまい。（　）サッカー選手だ。

| 选项 | 分析 |
| --- | --- |
| a 生まれるながらの | `生まれる` 这个动词形式不对；`ながら` 前面要用固定搭配的 `生まれながらの`。 |
| **b 生まれながらの** | `生まれながらの` 是固定说法，表示“天生就是……”；用来夸这位选手的天赋最自然。 |
| c 生まれたながらの | `生まれたながらの` 不是惯用表达，语法不成立。 |

**答案：b**

### Q2 祖母は（　）戦争中の思い出話を語った。

| 选项 | 分析 |
| --- | --- |
| a 涙ながらの | 这是修饰名词的形，像“哭着的……”，但后面这里接的是 `語った`，需要副词性表达。 |
| b 涙ながらで | `ながらで` 不是这里的固定用法，形式上不对。 |
| **c 涙ながらに** | `涙ながらに` 就是“含着泪、流着泪地”，和“讲述回忆”这个动作非常贴合。 |

**答案：c**

### Q3 久しぶりにふるさとを訪れた。（　）古い家がわたしを迎えてくれた。

| 选项 | 分析 |
| --- | --- |
| **a 昔ながらの** | `昔ながらの古い家` 是“保持着从前样子”的固定搭配，和“故乡里依旧老样子的房子”完全符合。 |
| b 昔ながらで | 这个形式后面不能直接这样修饰名词，接续不对。 |
| c 昔ながらに | `昔ながらに` 通常用作副词，不像这里要修饰 `古い家`。 |

**答案：a**

### Q4 立ち食いそば屋は、座らないで（　）そばを食べる店です。

| 选项 | 分析 |
| --- | --- |
| a 立って | 只有“站着”这个动作，少了和“吃荞麦面”并列的自然连贯感，不像固定搭配。 |
| b 立ちながらにして | 这个说法太硬，也不是常用搭配，放在这里不自然。 |
| **c 立ちながら** | “站着吃面”就是立食荞麦面的定义，`立ちながら` 最直接。 |

**答案：c**

## n1-l06-g05 〜きらいがある

**规则**：`辞書形/ない形/名の + きらいがある`；带有批评意味，表示“有……倾向”，多用于指出他人不好的习惯。

### Q1 今度の議長はどうも甘い言い方をするきらいがあると、みんなに（　）。

| 选项 | 分析 |
| --- | --- |
| **a 批判されている** | 既然说他有“说话太软”的倾向，那就表示别人对他有批评意见，句意完全合适。 |
| b 褒められている | 这和前面的负面倾向相反，语义冲突。 |
| c 喜ばれている | 也是正面评价，和 `きらいがある` 的批评语气不符。 |

**答案：a**

### Q2 うちの子は（　）きらいがある。

| 选项 | 分析 |
| --- | --- |
| a 動物をかわいがる | 这是偏正面的行为，不像 `きらいがある` 要指出的那种毛病或坏倾向。 |
| b 部屋を片付ける | 整理房间是好习惯，和“有不好的倾向”不搭。 |
| **c 物事を大げさに言う** | “把事情说得夸张”正是能被批评的坏倾向，最符合这个句型。 |

**答案：c**

### Q3 （　）頭で考えるだけで行動に移さないきらいがある。

| 选项 | 分析 |
| --- | --- |
| a わたしは | `きらいがある` 一般用于指出他人的倾向，不太用来自我评价。 |
| **b 弟は** | 用“弟弟”来指出“只想不做”的倾向，语气自然，也符合这个句型的常见指向。 |
| c 日本では | 地点做主语不成立，这里需要人做主语。 |

**答案：b**

### Q4 うちの夫は（　）きらいがある。

| 选项 | 分析 |
| --- | --- |
| a 趣味が多い | 只是说明爱好多，和“有某种不好的倾向”没有直接关系。 |
| b 趣味がない | 只是状态说明，不是“倾向”表达。 |
| **c 趣味を広げすぎる** | “把兴趣扩得太广”是可以批评的行为倾向，最适合 `きらいがある`。 |

**答案：c**

## n1-l07-g01 〜がてら

**规则**：名 / ます形 + がてら；表示“顺便做……”，前项是主要动作，后项是顺带动作。重点是两件事有主次关系，不是单纯并列。

### Q1 運動がてら（　）。

| 选项 | 分析 |
| --- | --- |
| a 運動場まで行ってきた | 这里只是“去运动场”，没有体现“顺便”的意思，前后动作主次不清。 |
| **b 犬の散歩に出かけた** | “运动时顺便去遛狗”最自然，既保留了主要动作 `運動`，也带出顺带做的事。 |
| c 毎日3キロ走っている | 这是习惯性的日常锻炼，不是“顺便”的动作。 |

**答案：b**

### Q2 （　）図書館に行った。

| 选项 | 分析 |
| --- | --- |
| **a 買い物がてら** | 先去买东西，顺便去图书馆，正是 `がてら` 的典型用法。 |
| b 本を借りがてら | `借りがてら` 结构别扭，顺序也不自然。 |
| c 本を探しがてら | 这个说法勉强能懂，但题目里更像是“办别的事顺路去图书馆”，因此不如 a 直接。 |

**答案：a**

### Q3 （　）車の展示場をのぞいた。

| 选项 | 分析 |
| --- | --- |
| a いい車を見つけがてら | “顺便找好车”意思太绕，且和后面的 `のぞいた` 搭配不如 a 直白。 |
| b 急ぎがてら | “赶时间顺便……”语义不清，也不符合日常表达。 |
| **c 遊びがてら** | “玩的时候顺便去看车展”最符合 `がてら` 的“顺路、顺便”语感。 |

**答案：c**

## n1-l07-g02 〜かたがた

**规则**：名 + かたがた；正式、郑重的“顺便兼带目的”。多用于报告、问候、拜访这类比较书面的场合。

### Q1 （　）かたがた近いうちに恩師を訪ねようと思っている。

| 选项 | 分析 |
| --- | --- |
| a 訪問 | 只是“访问”本身，和后面的“去拜访恩师”重复，放进句子里没有目的补充感。 |
| b 食事 | “吃饭”和拜访恩师之间没有那种郑重的兼带关系。 |
| **c 就職の報告** | “顺带报告就职消息去拜访恩师”很正式，完全符合 `かたがた` 的书面语气。 |

**答案：c**

### Q2 （　）かたがた、お宅に伺います。

| 选项 | 分析 |
| --- | --- |
| **a ごあいさつ** | `ごあいさつかたがた` 是很固定的敬语搭配，表示“顺带问候，顺便登门”。 |
| b 休日 | “假期”不是能和来访动作自然连上的名词。 |
| c おじゃま | `おじゃま` 本身就是“打扰、造访”，放在这里重复或冲突。 |

**答案：a**

### Q3 叔母が体調を崩したと聞いたので、見舞いかたがた（　）。

| 选项 | 分析 |
| --- | --- |
| a 手伝った | 这是完成式结果，和“见面慰问顺便……”的接续不搭。 |
| **b 手伝いに行った** | “去探病，顺便帮忙”是最自然的兼带目的表达。 |
| c 果物を贈った | 只是送礼物，少了“亲自前往”的动作感。 |

**答案：b**

## n1-l07-g03 〜かたわら

**规则**：名の / 辞书形 + かたわら；表示“在做本业的同时，也做另一件事”。核心是两种工作长期并行，不是偶尔顺手。

### Q1 彼は（　）のかたわらNPOの活動をしている。

| 选项 | 分析 |
| --- | --- |
| a 一人暮らし | 这不是“本业”，也没有“并行做别的事”的基础。 |
| **b 創作活動** | “在从事创作活动的同时做 NPO”很符合 `かたわら`。 |
| c 趣味 | `趣味` 太泛，和“本业并行”这个结构不够稳。 |

**答案：b**

### Q2 母は（　）かたわら翻訳の仕事もしている。

| 选项 | 分析 |
| --- | --- |
| **a 数学を教える** | “一边教数学，一边做翻译”是标准的并行工作。 |
| b 夕飯を作る | 做晚饭是家务，不像这里要表达的长期并行职业。 |
| c 音楽を聞く | 这只是日常行为，不像本业。 |

**答案：a**

### Q3 山中さんは市役所に勤めるかたわら（　）。

| 选项 | 分析 |
| --- | --- |
| **a 歌手としても活躍している** | “在市役所工作，同时也以歌手身份活跃”正是 `かたわら`。 |
| b いつも忙しくしている | 这是状态描述，不是另一个并行职业。 |
| c 家では年を取った親の世話をしている | 这是家务/私事，和句型想表达的“另有一业”不一样。 |

**答案：a**

## n1-l08-g01 〜ところを

**规则**：普通形 + ところを；表示“正在……的时候却……”，多用于致歉、打扰、感谢。

### Q1 （　）ところをおじゃましました。

| 选项 | 分析 |
| --- | --- |
| a お休み | 只是名词，没有形成 `お休みのところを` 的固定说法。|
| b お休みな | 语尾不自然，不能这样接。|
| **c お休みの** | `お休みのところを` 是固定搭配，表示“在您休息时打扰了”。|

**答案：c**

### Q2 すぐに（　）ところを遅くなってしまいまして、申し訳ありません。

| 选项 | 分析 |
| --- | --- |
| a お知らせする | 只是动词原形，不能直接和 `ところを` 接成“本该立刻通知却晚了”。|
| **b お知らせするべき** | `お知らせするべきところを` 表示“本该马上通知的时候却拖晚了”，语感最合适。|
| c お知らせした | 过去式不对应“本该……却……”的未完成感。|

**答案：b**

### Q3 お忙しいところを（　）。

| 选项 | 分析 |
| --- | --- |
| a 頑張ってくださいね | 这是鼓励，不是感谢或致歉，和前面的 `お忙しいところを` 不搭。|
| **b 手伝ってくださってありがとうございました** | 是标准的感谢表达，整句意思是“百忙之中还帮忙了，真谢谢您”。|
| c 何かお手伝いしましょうか | 这是提议帮忙，不是对对方已帮忙后的回应。|

## n1-l10-g05 〜ば〜で・〜なら〜で・〜たら〜たで

**规则**：ば/なら/たら + 同词 + で；表示“就算那样，也会有那样的麻烦”。

### Q2 大学に合格したら（　）入学金と学費の心配をしなければならない。
