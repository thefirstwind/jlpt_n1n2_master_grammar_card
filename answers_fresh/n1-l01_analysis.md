# N1 第1课 — 逐题分析（不参考旧答案）

## n1-l01-g01 〜が早いか

**规则**：动辞書形/た形 + が早いか；后接略带意外的事实；不接意志·请求。

### Q1 落書きをしていた子供たちは、わたしの顔を（　）が早いか、逃げていった。


| 选项       | 分析          |
| -------- | ----------- |
| a 見て     | て形，不接 が早いか  |
| **b 見る** | 辞書形，一见就逃，符合 |
| c 見るの    | の 名词化，语法不通  |


**答案：b**

### Q2 新しい化粧品が発売されるが早いか、（　）。


| 选项                | 分析    |
| ----------------- | ----- |
| **a 若い女性たちが殺到した** | 客观事实  |
| b 多くの女性に買ってほしい    | 希望，禁止 |
| c すぐに買ってください      | 请求，禁止 |


**答案：a**

### Q3 新幹線が走り出すが早いか、（　）。


| 选项                | 分析                      |
| ----------------- | ----------------------- |
| a みんなでお弁当を食べよう    | 意志，禁止                   |
| b みんなとても楽しそうだった   | 状态描写，可通但不如 c 的「紧接着发生」鲜明 |
| **c 彼はビールを飲み始めた** | 开车后立刻开始的意外动作，符合例句③④     |


**答案：c**

## n1-l01-g02 〜や・〜や否や

**规则**：动辞書形 + や/や否や；后接事实；不接意志·働きかけ。

### Q1 本格的に絵の勉強を（　）や、彼のデッサン力はみるみる上がった。


| 选项        | 分析              |
| --------- | --------------- |
| a している    | 进行时，不接 や        |
| **b 始める** | 一开始学就进步，辞書形     |
| c 続ける     | 继续学，与「刚开始就」的语感弱 |


**答案：b**

### Q2 サイレンが鳴るや、（　）。


| 选项             | 分析  |
| -------------- | --- |
| a すぐ飛び出そう      | 意志  |
| **b 消防車が出動した** | 事实  |
| c 校庭に集合しなさい    | 命令  |


**答案：b**

### Q3 太郎はアニメを1本見終わるや否や、（　）。


| 选项                          | 分析            |
| --------------------------- | ------------- |
| a 将来はアニメの仕事をしたいらしい          | 愿望            |
| b 今ではアニメがとても好きだ             | 一般状态，非「刚看完立刻」 |
| **c 別のアニメを借りにレンタルショップに行った** | 看完立刻去租下一部     |


**答案：c**

## n1-l01-g03 〜なり

**规则**：动辞書形 + なり；后接意外意志动作；主语通常三人称且前后一致。

### Q1 （　）は食卓に着くなり、ものすごい勢いで食べ始めた。


| 选项            | 分析          |
| ------------- | ----------- |
| a/b わたし/わたしたち | 一人称作主语不符合习惯 |
| **c 山川さん**    | 三人称         |


**答案：c**

### Q2 （　）なり、みんなその場から逃げ出した。


| 选项              | 分析                               |
| --------------- | -------------------------------- |
| a 大きな物音がする      | する 前需辞書形动作：「音がしたなり」才对，此处结构是「Vなり」 |
| **b その動物を一目見る** | 一看就逃                             |
| c だれかが大声で叫ぶ     | 叫ぶなり 可以，但 b 与例句②「入るなり」结构更平行      |


**答案：b**

### Q3 ゆき子は駅に着くなり、（　）。


| 选项              | 分析      |
| --------------- | ------- |
| a 切符を買った        | 普通行为    |
| b 電車の時刻表を見た     | 普通      |
| **c 駅員室に駆け込んだ** | 意外·急切动作 |


**答案：c**

## n1-l01-g04 〜そばから

**规则**：辞書形/た形 + そばから；刚…又…反复；多消极。

### Q1 あの店のパンは評判が良く、焼き上がるそばから（　）売れていく。


| 选项         | 分析        |
| ---------- | --------- |
| **a どんどん** | 刚出炉就不断卖出  |
| b 次第に      | 逐渐，非「刚…就」 |
| c そのうち     | 迟早，不对     |


**答案：a**

### Q2 わたしは（　）そばから人の名前を忘れてしまう。


| 选项        | 分析         |
| --------- | ---------- |
| a わかった    | 懂了≠刚懂就忘    |
| **b 聞いた** | 刚听到就忘      |
| c 知っている   | 状态，不接 そばから |


**答案：b**

### Q3 （　）そばから新しいのが欲しくなる。


| 选项          | 分析             |
| ----------- | -------------- |
| a 家を建てた     | 语境弱            |
| **b 服を買った** | 刚买又想要新的（消极反复）  |
| c テレビが壊れた   | 坏了想要新的，不是「刚…又」 |


**答案：b**

## n1-l01-g05 〜てからというもの（は）

**规则**：て形 + からというもの；之后状态一直持续；不用很近的过去。

### Q1 （　）というもの、体の調子がいい。


| 选项              | 分析     |
| --------------- | ------ |
| a 朝起きてから        | 太近     |
| **b たばこをやめてから** | 戒烟后一直好 |
| c さっき薬を飲んでから    | 太近     |


**答案：b**

### Q2 あの先生の話を聞いてからというもの、（　）。


| 选项                    | 分析   |
| --------------------- | ---- |
| **a 人生についていろいろ考えている** | 持续思考 |
| b 先生の著書を3冊買った         | 一次性  |
| c 先生の考えに同感した          | 一次性  |


**答案：a**

### Q3 会社を辞めてからというもの、（　）。


| 选项                  | 分析   |
| ------------------- | ---- |
| a 新しい仕事を始めた         | 一次性  |
| b 初めて外国旅行をした        | 一次性  |
| **c 毎日のように釣りをしている** | 持续状态 |


**答案：c**

## n1-l01-g06 〜にあって

**规则**：名 + にあって(も)；特殊情境下才/却发生。

### Q1 木村氏は（　）にあって、日々多忙なスケジュールをこなしている。


| 选项                | 分析          |
| ----------------- | ----------- |
| a 一国の指導者          | 可，但 b 更强调立场 |
| **b 一国の指導者という立場** | 在该立场上忙碌     |
| c 二つの仕事           | 不自然         |


**答案：b**

### Q2 母は（　）にあって、子供たちのことを心配している。


| 选项       | 分析      |
| -------- | ------- |
| **a 病床** | 卧病仍担心孩子 |
| b 仕事中    | 普通      |
| c 多忙な日常  | 不如病床反差强 |


**答案：a**

### Q3 このような緊急時にあっても、（　）。


| 选项              | 分析                     |
| --------------- | ---------------------- |
| a 彼は驚いている       | 符合紧急却惊慌，但 c 更体现「沉着」的反常 |
| b わたしは彼を探した     | 一次性                    |
| **c 彼は落ちついている** | 紧急时却镇定（にあっても 的反常）      |


**答案：c**
