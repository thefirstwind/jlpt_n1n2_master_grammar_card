# N1 第1课 — 逐题分析（不参考旧答案）

## n1-l01-g01 〜が早いか

**规则**：动辞書形/た形 + が早いか；后接略带意外的事实；不接意志·请求。

### Q1 落書きをしていた子供たちは、わたしの顔を（　）が早いか、逃げていった。


| 选项       | 分析 |
| -------- | ----------- |
| a 見て     | 这里是 て形，但「〜が早いか」前面要接辞書形或た形；用て形就断了句子，接不上后面的瞬间动作。 |
| **b 見る** | 辞書形能直接接「が早いか」。 |
| c 見るの    | 「の」把动词名词化了，后面不能再直接接「が早いか」；这里需要的是动作本身，不是名词短语。 |


**答案：b**

### Q2 新しい化粧品が発売されるが早いか、（　）。


| 选项                | 分析 |
| ----------------- | ----- |
| **a 若い女性たちが殺到した** | 这是紧跟“発売される”之后发生的客观事实，说明商品一上市就被抢购，符合「が早いか」后面接事实的特点。 |
| b 多くの女性に買ってほしい    | 这是说话人的希望，不是已经发生的事实；「が早いか」后面不能接这种意向表达。 |
| c すぐに買ってください      | 这是对听话人的请求命令，属于働きかけ，和「が早いか」要求的“紧接着发生的客观事实”不合。 |


**答案：a**

### Q3 新幹線が走り出すが早いか、（　）。


| 选项                | 分析 |
| ----------------- | ----------------------- |
| a みんなでお弁当を食べよう    | 这是群体的意志表达，明显是“要做什么”，而不是列车刚开动后马上发生的事实。 |
| b みんなとても楽しそうだった   | 这只是状态描写，能勉强通顺，但没有体现“走り出すが早いか”那种一发生立刻接动作的时间紧贴感，所以不如 c。 |
| **c 彼はビールを飲み始めた** | 列车刚开动，他就马上开始喝啤酒，后项是紧接着发生的实际动作，正是「が早いか」最典型的接法。 |


**答案：c**

## n1-l01-g02 〜や・〜や否や

**规则**：动辞書形 + や/や否や；后接事实；不接意志·働きかけ。

### Q1 本格的に絵の勉強を（　）や、彼のデッサン力はみるみる上がった。


| 选项        | 分析 |
| --------- | --------------- |
| a している    | 「や／や否や」要求前面用辞書形表示动作一发生就引出后项，进行时把动作拖长了，不符合这个瞬间衔接。 |
| **b 始める** | 「勉強を始めるや」表示一开始正式学习，后面就发生了能力提升；前后动作衔接紧密，语义最自然。 |
| c 続ける     | 「继续学」强调的是持续状态，不是“刚一开始”的瞬间；和这个句型要表达的时间关系不一致。 |


**答案：b**

### Q2 サイレンが鳴るや、（　）。


| 选项             | 分析 |
| -------------- | --- |
| a すぐ飛び出そう      | 这是说话人的意志或提议，属于主观表达；「鳴るや」后面应接已经发生的事实。 |
| **b 消防車が出動した** | 警报一响，消防车立刻出动，这是紧接着发生的事实，完全符合「や」的用法。 |
| c 校庭に集合しなさい    | 这是命令句，属于働きかけ，不能放在「や」后面；这个句型后面要接客观发生的事情。 |


**答案：b**

### Q3 太郎はアニメを1本見終わるや否や、（　）。


| 选项                          | 分析 |
| --------------------------- | ------------- |
| a 将来はアニメの仕事をしたいらしい          | 这是关于未来愿望的叙述，不是“看完立刻发生”的动作，因此和「や否や」的时间关系不合。 |
| b 今ではアニメがとても好きだ             | 这是现在的普遍状态说明，时间上太松散，没有体现“见终わるや否や”那种马上接着发生的动作。 |
| **c 別のアニメを借りにレンタルショップに行った** | 看完一部后立刻去租下一部，前后动作几乎无间隔，正好体现「や否や」的瞬间连续感。 |


**答案：c**

## n1-l01-g03 〜なり

**规则**：动辞書形 + なり；后接意外意志动作；主语通常三人称且前后一致。

### Q1 （　）は食卓に着くなり、ものすごい勢いで食べ始めた。


| 选项            | 分析 |
| ------------- | ----------- |
| a/b わたし/わたしたち | 「なり」常接三人称或客观动作主体；用第一人称会显得像自述心理，不太符合这个句型的叙述习惯。 |
| **c 山川さん**    | 第三人称人物一坐下就开始大口吃饭，最符合「着くなり、…始めた」这种“刚…就…”的典型场景。 |


**答案：c**

### Q2 （　）なり、みんなその場から逃げ出した。


| 选项              | 分析 |
| --------------- | -------------------------------- |
| a 大きな物音がする      | 这里不是“做了一个动作后立刻发生另一个动作”，而且「物音がする」本身也不够像一个可直接串接「なり」的动作起点。 |
| **b その動物を一目見る** | “一眼看见那只动物就逃跑”是典型的「一…就…」结构，前后动作非常紧，最符合「なり」的用法。 |
| c だれかが大声で叫ぶ     | 语法上并非完全不能接，但题目想要的是“某动作一发生就立即逃跑”的最贴切搭配；和 b 相比，少了一眼触发逃跑的那种直接感。 |


**答案：b**

### Q3 ゆき子は駅に着くなり、（　）。


| 选项              | 分析 |
| --------------- | ------- |
| a 切符を買った        | 只是进入车站后的普通动作，没有“刚到站就立刻发生的出乎意料动作”这层时间关系。 |
| b 電車の時刻表を見た     | 这是一般性的查看信息，不是“着くなり”后面那种立刻、急切、出人意料的动作。 |
| **c 駅員室に駆け込んだ** | 刚到车站就冲进站员室，动作急、间隔短，而且带有突发感，最符合「着くなり」的语感。 |


**答案：c**

## n1-l01-g04 〜そばから

**规则**：辞書形/た形 + そばから；刚…又…反复；多消极。

### Q1 あの店のパンは評判が良く、焼き上がるそばから（　）売れていく。


| 选项         | 分析 |
| ---------- | --------- |
| **a どんどん** | “焼き上がるそばから”强调刚出炉马上被买走，后面自然接表示持续推进的副词「どんどん」，语义最顺。 |
| b 次第に      | 「次第に」是慢慢变化，不是“刚…就”的连续反应，时间感完全不同。 |
| c そのうち     | 「そのうち」表示过一会儿、迟早，并不强调动作一发生就马上跟着另一个结果，不合这里的语气。 |


**答案：a**

### Q2 わたしは（　）そばから人の名前を忘れてしまう。


| 选项        | 分析 |
| --------- | ---------- |
| a わかった    | “懂了”本身不是错误，但题目想表达的是“刚听到就忘”，所以需要更直接的触发点，而不是理解完成这个结果。 |
| **b 聞いた** | 一听到名字就忘，正是「そばから」那种“刚做完一件事，紧接着就反复出问题”的语感。 |
| c 知っている   | 这是持续状态，不是动作起点；「そばから」要接“刚…就…”的动态场景，不能接单纯状态。 |


**答案：b**

### Q3 （　）そばから新しいのが欲しくなる。


| 选项          | 分析 |
| ----------- | -------------- |
| a 家を建てた     | 盖房子后马上又想要新的东西，语义跳得太大，缺少“刚得到就不满足”的那种反复感。 |
| **b 服を買った** | 刚买完衣服又想要新的，正好表现“得到了也不满足、马上又想要别的”的消极反复，最符合「そばから」。 |
| c テレビが壊れた   | “坏了想要新的”是正常因果，不是“刚…又…”的反复；缺少「そばから」要求的时间贴近感。 |


**答案：b**

## n1-l01-g05 〜てからというもの（は）

**规则**：て形 + からというもの；之后状态一直持续；不用很近的过去。

### Q1 （　）というもの、体の調子がいい。


| 选项              | 分析 |
| --------------- | ------ |
| a 朝起きてから        | 这里虽然也是“从…之后”，但太近的过去只是短时间经过，不足以承接「からというもの」那种持续到现在的长期状态。 |
| **b たばこをやめてから** | 戒烟之后，身体状态一直变好，符合“从那以后一直如此”的持续变化。 |
| c さっき薬を飲んでから    | “刚刚吃药以后”是非常近的时间点，通常只是短暂说明，不像「というもの」强调的长时间持续。 |


**答案：b**

### Q2 あの先生の話を聞いてからというもの、（　）。


| 选项                    | 分析 |
| --------------------- | ---- |
| **a 人生についていろいろ考えている** | 听了老师的话以后一直在思考人生，符合“自那以后持续受到影响”的用法。 |
| b 先生の著書を3冊買った         | 这是一次性的购买动作，不能体现听完之后状态持续改变，所以不合「からというもの」。 |
| c 先生の考えに同感した          | “同感了”只是瞬间的态度判断，不足以表现后面持续展开的状态变化。 |


**答案：a**

### Q3 会社を辞めてからというもの、（　）。


| 选项                  | 分析 |
| ------------------- | ---- |
| a 新しい仕事を始めた         | 开始新工作是单次动作，不能说明“辞职之后一直怎样”的持续状态。 |
| b 初めて外国旅行をした        | 这也是一次经历，不是持续性的生活状态。 |
| **c 毎日のように釣りをしている** | “几乎每天都在钓鱼”正是从辞职后一直延续下去的生活变化，很符合这个句型。 |


**答案：c**

## n1-l01-g06 〜にあって

**规则**：名 + にあって(も)；特殊情境下才/却发生。

### Q1 木村氏は（　）にあって、日々多忙なスケジュールをこなしている。


| 选项                | 分析 |
| ----------------- | ----------- |
| a 一国の指導者          | 语义上有“身份”意味，但还不够明确；后面接「にあって」时，最好把特殊处境或立场说得更清楚。 |
| **b 一国の指導者という立場** | 明确指出“作为一国领导者的立场”，再接忙碌的日程，就能突出这个特殊身份下仍要承担大量事务。 |
| c 二つの仕事           | “两个工作”只是数量说明，不是能够引出特殊处境的名词短语，和「にあって」的搭配不自然。 |


**答案：b**

### Q2 母は（　）にあって、子供たちのことを心配している。


| 选项       | 分析 |
| -------- | ------- |
| **a 病床** | 卧病在床时还担心孩子，正是“处于特殊困难状态中却仍然…”的典型场景。 |
| b 仕事中    | “工作中”只是一般状态，不带困难、逆境或特别环境的色彩，和这个句型的特殊环境语感不够吻合。 |
| c 多忙な日常  | 忙碌日常也能接，但题目想强调的不是普通忙，而是“在艰难处境中仍然担心孩子”的反差感，所以不如 a。 |


**答案：a**

### Q3 このような緊急時にあっても、（　）。


| 选项              | 分析 |
| --------------- | ---------------------- |
| a 彼は驚いている       | 紧急时惊慌当然也说得通，但「にあっても」更强调“即使处于这种环境下仍然怎样”，最需要的是与紧急环境形成反差的状态。 |
| b わたしは彼を探した     | 这是单纯发生的动作，没有把“紧急环境中的反差结果”表现出来，因此和句型重点不一致。 |
| **c 彼は落ちついている** | 在紧急时刻还能保持镇定，正好体现“身处这种特殊情况却不受影响”的反差，是这个句型最核心的意思。 |


**答案：c**
